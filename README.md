# LIFE4141_2526

Repository for materials for LIFE4141 practicals and coursework.

Each session directory contains a README.md markdown file with instructions on how to get started.

Where relavent, session directories also contain a conda yaml file which can be used to install all the software that you'll need in the session with the following command: (replacing `file.yaml` with the name of the specific yaml file)

```
conda env create -f file.yaml
```
